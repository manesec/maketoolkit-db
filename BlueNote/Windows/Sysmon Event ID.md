# Sysmon event ID

```
Event ID 1: Process Creation - 记录进程的创建事件，包括命令行参数、进程 ID 等信息。
Event ID 2: File Creation Time Changed - 记录文件创建时间的更改。
Event ID 3: Network Connection - 记录网络连接事件，包括源和目标 IP 地址、端口等。
Event ID 4: Syscall - 记录系统调用事件。
Event ID 5: Process Termination - 记录进程终止事件。
Event ID 6: Driver Loaded - 记录驱动程序加载事件。
Event ID 7: Image Loaded - 记录加载的图像（如 DLL）事件。
Event ID 8: CreateRemoteThread - 记录远程线程创建事件。
Event ID 9: RawAccessRead - 记录原始访问读取事件。
Event ID 10: ProcessAccess - 记录进程访问事件。
Event ID 11: FileCreate - 记录文件创建事件。
```
