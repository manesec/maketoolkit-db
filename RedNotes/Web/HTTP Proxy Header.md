# Http Proxy Header

Add header: 

+ `X-Forwarded-For: <ip-address>`
+ `X-Originating-IP: <ip-address>`
+ `X-Remote-Addr:<ip-address>`
+ `X-Remote-Ip: <ip-address>`

## Firefox Plugin
